# URP高校教务管理系统实用工具

用于帮助使用URP高校教务管理系统（University Resource Planning）的高校学生便捷操作的小工具。

## Description

- 本项目可快速完成教学评估，免去每个问卷需要三分钟才能提交的时间，也避免了不可填写重复项目的反人类特性。此外，还可快速查询最新学期的成绩和绩点。

## Features

- 🔐 登录URP高校教务管理系统
- 🤖 自动识别验证码/手动填写验证码
- 📝 自动获取未评估课程列表
- ⚡ 批量完成教学评估
- 📈 快捷查询最新学期的成绩和绩点

## Environment

- Python 3.14+
- 可访问URP高校教务管理系统的网络环境

### Dependencies

- **requests**: HTTP 请求
- **beautifulsoup4**: HTML 解析
- **ddddocr**: OCR识别

## Usage

1. 运行程序

若要自动识别验证码登录，请运行 `auto.py` 文件。
```bash
python auto.py
```

若要手动填写验证码登录，请运行 `main.py` 文件。
```bash
python main.py
```

若要查询成绩和绩点，请运行 `results.py` 文件。
```bash
python results.py
```

2. 输入你的学号和密码

## Configuration

### 修改目标服务器

如果你的 URP 系统地址不是 `http://192.168.16.207:9001`，需要在代码中修改相应的 URL

### 自定义评估选项

评估选项在 `submit_evaluation` 方法的 `evaluation_data` 字典中，可根据实际需求修改评分。

## Common Issues

### Q1: ddddocr安装失败

**可能原因**：
- Python 版本不支持
- 环境变量配置错误
- 网络问题

**解决方法**：
1. 确保 Python 版本为 3.13+
2. 检查环境变量配置是否正确
3. 确认网络连接正常，尝试重新安装
4. 如果问题持续存在，使用手动填写验证码的版本

```bash
python main.py
```

### Q2: 评估提交失败

**可能原因**：
- 网络连接问题
- Session 过期
- 服务器地址配置错误

**解决方法**：
1. 检查网络连接，需要在学校内网
2. 重新运行程序重新登录
3. 确认代码中的服务器地址是否正确
